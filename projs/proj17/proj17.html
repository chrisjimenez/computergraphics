Creating a Story Using the HTML5 Canvas

The main idea about this assignment was to have fun and to use what we have learned about the HTML5 canvas and matrix operations to create a little story! So I created a little story about a little triangular character being chased by a creature.

The character, enemy, and background scene turn at the same rate(which is dependent on time) and at the same angle. So as our friend gets chased down the sidewalk, we can see the scenery behind him! The bright light above represents the sun, which moves with the background scene.

I had some fun with this assignment! But it was surprisingly difficult positioning everything on the canvas using the matrix operations.
